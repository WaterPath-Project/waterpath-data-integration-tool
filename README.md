## 🚀 Quick start

1.  **Clone this repo.**

    Access a CLI to clone this repository.

    ```shell
    git clone https://github.com/WaterPath-Project/waterpath-data-integration-tool.git
    ```

2.  **Start developing.**

    Navigate into the cloned repo's directory and start it up.

    ```shell
    cd waterpath-data-integration-tool/
    npm install
    npm start
    ```

3.  **Open the code and start customizing!**

    Your site is now running at http://localhost:3000!

    Edit any file in `app/` to see the site updated in real-time!
